## flex 特性

- 在 `flex` 區塊內，第一層區塊寬度由 `flex` 統一處理
- `flex` 預設不會自動換行

## css 優先權

- 宣告越嚴謹，優先權越高
- 同樣優先權時，後者覆蓋前者

```css
.flex-item.red {
    background: red;
}

.flex-item {
    background: yellow;
}
```

## flex 超出換行行為

屬性 `flex-wrap`

- `nowrap` 不換行，壓縮現有區塊寬度
- `wrap` 換行，如 `flex` 本身有設定高度時，可能會破版
    - 使用 `min-height` 可以排除此問題
- `wrap-reverse` 換行且上下對調
