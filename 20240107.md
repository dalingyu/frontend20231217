## CSS 宣告格式

### 使用 `<style></style>` 標籤

***正確方式***

```html
<style>
    h1 {
        color: red;
        background-color: yellow;
        font-size: 200px;
    }
</style>
```

***錯誤方式***

```html
h1 {
    color: red;
    background-color: yellow;
    font-size: 200px;
}
```

樣式會直接顯示在網頁上
